Как собирать команды
Шаг 1
Впишите голосовой триггер построчно. Пример: открой телеграм или режим кино.
Шаг 2
Выберите действие: только ответ, shell-команда или HTTP/webhook для умного дома.
Шаг 3
Выберите озвучку: TTS для живого ответа или свой WAV, если нужен фирменный отклик.
Фраза: открой браузер -> shell -> start chrome
Фраза: включи свет в спальне -> http -> Home Assistant webhook
Фраза: спокойной ночи -> speak -> выключаю дом, сэр
Фраза: режим тревоги -> wav -> siren.wav
